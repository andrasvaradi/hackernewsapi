This is a single page application using the Hacker News public API https://github.com/HackerNews/API.

The app loads 30 of the 'top 500 news stories' from the api.

Each story displaying the following information:
- Title(clickable)
- Url
- Upvote score
- Author
- Time passed since posting

At the bottom of the list you can click on 'MORE' which will load an other 30 news stories, recurring.

Extra features:
- state is managed via Redux
- added some basic styling
- added 2 types of sorting options

Technologies used:
React, Redux, Material UI, Sass
